內容如下：

#include <python.h>

#ifdef __cplusplus
extern "C" {
#endif

static PyObject* sayHello(PyObject* self, PyObject* args);
static PyObject* sayGoodBye(PyObject* self, PyObject* args);

static PyMethodDef awin_pyMethods[] =
{
    { "awin_func1", sayHello, METH_VARARGS, "A function to print hello." },
    { "awin_funGoodBye", sayGoodBye, METH_VARARGS, "A function to print goodbye." },
    { NULL, NULL, 0, NULL }
};

static struct PyModuleDef awin_pymodule = {
    PyModuleDef_HEAD_INIT,
    "awin_py",   /* name of module */
    "A module to wrapper c code", /* module documentation, may be NULL */
    -1,       /* size of per-interpreter state of the module, or -1 if the module keeps state in global variables. */
    awin_pyMethods
};

PyMODINIT_FUNC PyInit_awin_py(void)
{
    PyObject *m;

    m = PyModule_Create(&awin_pymodule);
    if (m == NULL)
        return NULL;

    return m;
}

static PyObject* sayHello(PyObject* self, PyObject* args)
{
    /*
     *  Accept argument: name(string)
     *  Retrun: None
     *  Python Code:
     *      awin_py.sayHello("Awin")
     */

    char* name;

    if (!PyArg_ParseTuple(args, "s", &name))
        return NULL;

    printf("Hello: %s\n", name);

    Py_RETURN_NONE;
}

static PyObject* sayGoodBye(PyObject* self, PyObject* args)
{
    /*
    *  Accept argument: hour(int), minute(int)
    *  Retrun: hour + min(int)
    *  Python Code:
    *      awin_py.sayGoodBye(21, 30)
    */

    int hour, min;

    if (!PyArg_ParseTuple(args, "II", &hour, &min))
        return NULL;

    printf("Goodbyte at %d:%d\n", hour, min);

    return PyLong_FromLong(hour + min);
}

#ifdef __cplusplus
}
#endif

主要的部分有三個：
1. PyModuleDef，這是定一本module的名字與function structure。
2. PyMethodDef，可以被Python呼叫的function都定義在這邊。第一個欄位是Python內會看到的method name，第二個欄位是C的function name，第三個欄位就填METH_VARARGS，第四個欄位是method的說明。
3. 最後是PyInit_awin_py，注意這裡的function name必須是PyInit_ + module name，本例的module name是awin_py，所以init function的名字就必須是PyInit_awin_py()。

最後是C function的內容。
在C function內是用PyArg_ParseTuple()來找出Python丟下來的參數，詳細的參數列表可以看Parsing arguments and building values，以sayHello()來說 ，接受一個字串參數，所以用s來將Python丟下來的參數放到name變數裡。因為這function沒什麼藥return給python的，所以最後用Py_RETURN_NONE。

sayGoodBye()則是為了示範接收兩個int參數，所以用了兩個I。最後回傳一個整數給Python，所以用了PyLong_FromLong()來將整數轉為Python Object回傳。

PyXXX_FromXXX之類的function很多，可以參考Integer Objects。
